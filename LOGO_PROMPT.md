# Logo Prompt — ScribeTube (Bulk YouTube Transcriber)

## Primary prompt

```
A modern, minimalist app icon logo for "ScribeTube", a bulk YouTube
transcription tool.
Central concept: a red YouTube-style play button transforming into flowing
lines of text / subtitle bars on the right, symbolizing video-to-text
conversion. Include subtle stacked/duplicated layers behind the play button
to convey "bulk" / batch processing. Clean flat vector design, bold simple
shapes, rounded square icon with soft gradient background (deep red to dark
charcoal), centered composition, high contrast, crisp edges, no text labels,
professional tech branding, 1:1 aspect ratio, app store icon style.
```

## Variations

### Neon / dark mode
```
App icon for a YouTube bulk transcription tool: glowing red play button
dissolving into neon subtitle lines on a dark charcoal background, cyberpunk
tech aesthetic, soft outer glow, flat vector, rounded square, 1:1, no text.
```

### Waveform-to-text motif
```
Minimal logo: an audio waveform on the left morphing into clean text lines on
the right, with a small red play triangle accent, stacked layers for batch
processing, flat design, gradient red-to-slate background, rounded square app
icon, high contrast, 1:1, no text labels.
```
