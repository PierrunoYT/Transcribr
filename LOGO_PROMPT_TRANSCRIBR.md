# Logo Prompt — Transcribr

A fresh icon concept for the rebranded app: a **general** bulk audio & video
transcriber (YouTube + local file uploads), not YouTube-specific.

## Primary prompt

```
A modern, minimalist app icon for "Transcribr", a bulk audio-and-video
transcription tool. Central concept: a clean sound waveform on the left
flowing rightward and morphing into neat horizontal lines of text /
subtitle bars, symbolizing speech-to-text. Add subtle stacked/duplicated
layers behind it to convey batch "bulk" processing. Flat vector design,
bold simple shapes, rounded square icon, soft gradient background (deep
indigo to slate blue) with a single bright accent color, centered
composition, high contrast, crisp edges, no text labels, professional
tech branding, 1:1 aspect ratio, app store icon style.
```

## Variations

### Letter mark
```
Minimal app icon: a stylized lowercase letter "t" formed from a sound
waveform that resolves into text lines, monochrome with one teal accent,
flat vector, rounded square, soft gradient background, 1:1, no extra text.
```

### Document + waveform
```
App icon for a transcription tool: a clean document/page with text lines,
overlaid by a glowing audio waveform crossing it diagonally, modern flat
design, gradient violet-to-blue background, rounded square, high contrast,
1:1 aspect ratio, no text labels.
```

### Dark / neon
```
Sleek dark-mode app icon: neon waveform on the left dissolving into glowing
subtitle text lines on the right, charcoal background, cyberpunk tech
aesthetic, soft outer glow, flat vector, rounded square, 1:1, no text.
```
